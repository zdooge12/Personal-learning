Initialization

  • Stored in .HTML files
  • Start Doc with <!DOCTYPE html>
  • Include all HTML code within <HTML> </HTML>
  • Include machine readable page data with <head> </head>
  • Show visible content within <body> </body>

Common Elements "Tags" or "Attributes"

All these items commonly use the 'class' and 'id' attributes

Element Description
<div> </div> Div "Division"is commonly used for dividing sections or containers
<h> </h> h items are used for headers within the page. H1 is highest level and h6 is lowest
<p> </p> 'p' items are for paragraphs of standard text.
<br /> 'br' is for break and conveys a line break. Usually used within 'p' tags

Tables

Element Description
<table> </table> These tags initialize a table
<thead> </thead> Indicates a table header row. Usually used for the first row.
<tr> </tr> Indicates a new row of data.
<td> </td> Indicates a 'cell' of data.
<tfoot> </tfoot> Indicates a footer row in a table

Lists

Element Description
<ul> </ul> Unordered List. Initializes for a bullet style list.
<ol> </ol> Ordered List. Initializes using numeric for the list items.
<li> </li> List Item. Each item in the list uses these tags.

Reserved Characters

HTML has some special characters that make up the language; they are the reserved character. The reserved characters in HTML are < , > , " and &. The browser will never display them since they have some meaning in HTML.
HTML entities are pieces of text used to display reserved characters, invisible characters(like space) and other non-keyboard characters.
Entities begin with the "&" followed by "entity name" or "entity number" and end with the ";" ie. &entity-name; or &entity-number;
